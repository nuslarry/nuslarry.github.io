Projects

Relational Database Management System

This is a database management system that supports basic SQL operations including Selection, Projections, Joins and Aggregate operations.The system consists of five key modules. The first module is page file manager, which handles data reading and writing. Every piece of data in this system is stored in database pages. The second module is record based manager, which handles the structure of a page. The third module is index manager, which uses unclustered B+ balanced tree to create indexes for records. The forth module is relation manager, which extracts the information of the records that match a specific condition in a page. The fifth module is search engine. The engine is responsible for extracting information of records that match several conditions simultaneously.
