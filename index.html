and got much recognition.

Cheng Liang has rich experience in software development. His most recent project is a database
management system written in C++ which supports basic SQL operations. Besides this, last year he also
implemented a desktop canlendar app in Java with three other teammates. The canlendar suppports
agenda plannning functionalities like customized searching and recurring events. Furthermore, he also
has experience in real-time web application. He developed a web chat application that supports real
time bi-directional communication between clients and servers using express.js and socket.io and
deployed the app to the internet.

Cheng Liang is enthusiastic about practical, challenging projects and tasks. Furthermore, he is

Cheng Liang is also active in extracurricular activities. He was a member of cake club and they

He is a strong candidate for a software engineer position and can contribute a lot to the company
